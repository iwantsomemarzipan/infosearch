# Репозиторий для ДЗ 1

Корпус текстов представляет собой тексты отзывов на духи (1800 всего), собранные с сайта [randewoo.ru](https://randewoo.ru) (причина — я фанат парфюмерии :) ).

Файлы с кодом:
- 'crawler.py' — краулер для сбора отзывов
- 'preprocessing.py' — предобработка собранных отзывов (см. 'reviews_preprocessed.csv')
- 'search.py' — пункты 3-5 по условию ДЗ
- 'manual_indices.py' — пункт 6, а также 1 на оценку 8 по условию ДЗ
